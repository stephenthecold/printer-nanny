Recovery checklist
  1. Take a fresh backup on the source instance.
  2. Stand up the destination: same major versions of central + Postgres.
  3. Run alembic upgrade head -- restore overlays into a working schema.
  4. Upload the backup here and type RESTORE.
  5. Click Update all agents on /manage/agents so they refresh against the new central URL/key.
